# Mobile App Assets

This directory should contain the following assets for the Expo app:

## Required Assets

1. **icon.png** - App icon (1024x1024px)
2. **splash.png** - Splash screen (2048x2048px) 
3. **adaptive-icon.png** - Android adaptive icon (1024x1024px)
4. **favicon.png** - Web favicon (48x48px)

## Creating Placeholder Assets

For development, you can create simple placeholder images using any image editor.

### Recommended Colors
- Primary: #4F46E5 (Indigo)
- Background: #FFFFFF (White)

### Placeholder Script (optional)
You can use any online tool to generate placeholder images:
- https://placeholder.com/
- https://www.canva.com/

## Production Assets

For production, create professional assets that:
- Have clear branding
- Are properly sized for each platform
- Use appropriate padding for adaptive icons
